# Execution in Development

1. Clone the repository
2. Execute
```
yarn install
```

3. Install or to have Nest CLI installed
```
npm install -g @nestjs/cli
```

4. Start the database
```
docker-compose up -d
```

## Stack used
* MongoDB
* Nestjs
